# NumOptions
`NumOptions` is used inside of [Num's Constructor](/docs/num#constructor) to give extra information about a number. They are especially important when converting a non-base 10 number to another base.
###### 
| Name | Description | Type | Example |
| :--- | :--- | :--- | :--- |
| num | The number being wrapped by `Num`. | [String](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String)/[Number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) | "1a" |
| base? | The base of `num`. | [Number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) | 16 |
